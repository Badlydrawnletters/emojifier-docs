# Convertwumpus <img src="images/wumpusbeach.svg" width="50" height="50" />
---
### Description
This command is used to convert text into Wumpus letter emojis
### Usage
```
!!convertwumpus <text>
```
### Permission Required
Anyone can use this command, unless they are blacklisted globally
